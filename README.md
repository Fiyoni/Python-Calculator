# Python Calculator

## Description
A console-based Python calculator application that performs basic arithmetic operations including addition, subtraction, multiplication, division, modulus, and power. Handles invalid input and zero-division errors. Built to strengthen Python fundamentals and logical problem-solving skills.

## Features
- Addition, Subtraction, Multiplication
- Division with zero check
- Modulus and Power operations
- Menu-driven interface for easy usage

## Language & Tools
- Python

## Skills Learned
- Python basics: variables, loops, conditionals, functions
- Input validation
- Logical thinking for calculator operations
